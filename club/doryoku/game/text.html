データ構造

グローバル状態管理 (gameState)

player
プレイヤーの全状態を保持するオブジェクト。位置・カメラ角度・HP・速度・接地状態・ダッシュフラグなどをまとめて管理。
player: { x, y, z, // 座標 pitch, yaw, // 視点角度 health: 100, velocity: {x,y,z}, // 速度ベクトル isGrounded, isDashing, lastJumpTime }
inventory / ammo
8スロットのインベントリ配列。スロット0=弓、スロット1=矢は固定。2〜7がブロック・アイテム用スロット。
inventory: [ { type: 'bow', count: 1 }, { type: 'arrow', count: 20 }, { type: 'block', blockId, count }, { type: 'empty' }, // ...スロット7まで ]
arrows / enemies
飛行中の矢と敵エンティティの配列。毎フレームupdateArrows・render内でメッシュ数をこの配列の長さに合わせて動的に調整。
arrows: [{ x, y, z, // 現在位置 vx, vy, vz, // 速度 damage, hit, life, ownerId }]
blockHealth / droppedItems
ブロックのHPはMapで "x,y,z" キーで管理。矢が当たった石ブロックはHP50%でひび石化。ドロップアイテムも配列で管理。
blockHealth: new Map() // キー: "x,y,z" // 値: HP数値 (石 = 60) droppedItems: [{ x, y, z, vx, vy, vz, mesh, life: 600 }]
